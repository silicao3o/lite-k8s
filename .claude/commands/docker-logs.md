# Docker Logs Command

Docker 컨테이너 로그를 조회하고 분석합니다.

## Instructions

지정된 컨테이너의 최근 로그를 조회하고 분석합니다.

## Parameters

컨테이너 이름 또는 ID: $ARGUMENTS

## Tasks

1. 지정된 컨테이너의 최근 로그 100줄 조회
2. 에러 또는 경고 메시지 필터링
3. 로그 패턴 분석 및 이슈 요약

## Command

```bash
docker logs --tail 100 <container_name>
```
