# Code Review Command

코드 품질을 검사하고 개선점을 제안합니다.

## Review Checklist

다음 항목들을 검토하세요:

### 코드 품질
- [ ] 중복 코드가 없는지 확인
- [ ] 함수/메서드가 단일 책임을 가지는지 확인
- [ ] 네이밍이 의도를 명확히 표현하는지 확인
- [ ] 불필요한 복잡성이 없는지 확인

### 테스트
- [ ] 테스트 커버리지가 충분한지 확인
- [ ] 테스트가 의미 있는 케이스를 다루는지 확인
- [ ] 엣지 케이스가 처리되는지 확인

### 보안
- [ ] 입력 검증이 적절한지 확인
- [ ] 민감한 정보가 노출되지 않는지 확인

## Target

리뷰 대상: $ARGUMENTS

변경된 파일들을 분석하고 위 체크리스트에 따라 리뷰 결과를 제공하세요.
