# TDD Cycle Command

TDD Red-Green-Refactor 사이클을 실행합니다.

## Instructions

1. **Red Phase**: 현재 작업 중인 기능에 대한 실패하는 테스트를 작성하세요
2. **Green Phase**: 테스트를 통과시키기 위한 최소한의 코드만 구현하세요
3. **Refactor Phase**: 테스트가 통과하면 코드 구조를 개선하세요

## Workflow

- plan.md에서 다음 미완료 항목을 찾으세요
- 해당 항목에 대한 테스트를 먼저 작성하세요
- 테스트 실행 후 실패 확인
- 테스트 통과를 위한 최소 구현
- 모든 테스트 통과 확인
- 필요시 리팩토링 수행
- 구조적 변경과 동작 변경을 별도 커밋으로 분리
